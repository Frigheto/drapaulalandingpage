# Dra. Paula & Dr. Breno Amorim - Advocacia Trabalhista

Landing page desenvolvida para o escritório de advocacia trabalhista da Dra. Paula Marcula e Dr. Breno Amorim, com foco em atendimento 100% online em todo o Brasil e presencial em Petrolina - PE.

## 🚀 Tecnologias Utilizadas
- HTML5
- CSS3 (Variáveis CSS, CSS Animations)
- Design responsivo
- Javascript Vanilla (Manipulação de DOM para FAQ e Smooth Scroll)

## 🎨 Características do Projeto
- **Design Elegante:** Tema escuro premium com contrastes em dourado (`--accent-gold`).
- **Showcase 3D:** Exibição realista da interface em múltiplos dispositivos (Monitor, Laptop, Tablet, Smartphone) usando apenas CSS e transformações 3D.
- **Micro-interações:** Animações CSS como `fadeInUp`, `glowPulse` e `shimmer`.
- **FAQ Interativo:** Componente de perguntas e respostas expansível.
- **Responsividade:** Media queries garantem adaptação perfeita em dispositivos móveis e tablets.
- **Botão Flutuante do WhatsApp:** Atalho rápido para atendimento.
